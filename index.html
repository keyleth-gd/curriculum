# Projetos

# Stateless

Um jogo analógico competitivo inspirado em mercenários, onde cada jogador controla um bando de mercenários que precisa realizar contratos com o objetivo de se tornar o bando mais famoso. O jogo se baseia em mecânicas de Construção de Deck, Missões e Leilão. Projeto desenvolvido em grupo dentro da Faculdade Mèliés, durante esse projeto meu principal papel foi Game Designer.

# Conectando a Hora do Break

Um jogo onde um estagiário se vê na missão de melhorar seu ambiente de trabalho ajudando os funcionários a lidar com seus problemas pessoais. Ele consegue isso através de "combates" que são conversas em um sistema de turnos onde o personagem usa argumentos para modificar as emoções dos funcionários. Projeto desenvolvido em grupo para I Méliès GameJam, onde meu cargo foi Game Designer.

# Does it feel like home?

Um jogo acompanhando a processo de Isaac de transformar sua nova casa em um lar. O jogo é um point-and-click onde o jogador deve encontrar itens que irão ativar a memória de isaac. Projeto desenvolvido durante uma GameJam promovida pela professora Lia na disciplina Engenharia de Software na Faculdade Méliès. Nesse projeto trabalhei com Game Design e Programação.

# I Miss You

I Miss You é uma Visual Novel desenvolvida para a Women Game Jam 2020 que conta a história de Sissi e sua vida longe de suas amigas. O projeto foi desenvolvido junto com mais três pessoas, neste projeto atuei com Programação e Game Design.

# Alquixinim

Alquixinim é um jogo 2D de ação onde dois jogadores competem pela pedra filosal, eles devem atravessar um labirinto repleto de armadilhas tendo a sua disposição habilidades como dash, disparos de energia e alguns power-ups encontrados pelo mapa.

# Synapses

Synapses é um jogo de storytelling e puzzle onde uma garota acorda em uma sala de hospital com um médico e um policial, entretando, ela perdeu suas memórias, agora você precisa realizar sinapses para recuperar essas memórias e responder as perguntas que estão sendo feitas. Esse jogo foi desenvolvido para a Global Game Jam 2021 onde atuei como Game Designer.

# Oasis

Oasis é um jogo 3D de aventura, onde o jogador comanda Lina, uma sobrevivente do deserto que está presa em um ambiente hostil e precisa encontrar a herança de seu povo e uma forma de continuar sua jornada.
